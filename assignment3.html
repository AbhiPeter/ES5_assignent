what is the difference between factory and constructor:-

    - the basic difference is that constructor function is used with  new keyword
        var objtfromconstructor= new constructor funtion();

    -factory function is like a regular function without new keyword
        var objfrom constructor=constructor function();
        for factory it would need to return a  new instance of some object.
    -Any function can return a new object when its not constructor or class its called as factory function
