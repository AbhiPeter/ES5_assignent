1)how can we create class in javascript?
    - In ES5 we can create a class by constructor.
    -with help of the constructor we can creat a class in ES5.

    function Person(firstName,lastName)
    {
        this.firstName=firstName;
        this.lastName=lastName;

    }

    
    you can also extended the classes that means inherit it by using prototype

    here we can inherit the class by using prototype:-

    Person.prototype.getFullName=function()
    {
        return this.firstName + ' ' + this.lastName;
    }
